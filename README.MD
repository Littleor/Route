# 同时上内外网解决办法(MAC)
> 鉴于MAC上无法执行bat文件,所以把原来的win版的bat文件改成了MAC版,分享给每一个需要内外网同时使用的MAC用户 
## 使用说明 
> 1. 下载sh文件  
> 2. 打开终端(Terminal)  
> 3. cd到sh文件目录下 
> 4. 执行 ./route.sh 
> 5. 阅读说明3s之后找到找到broadcast后面的172或者202开头的IP地址复制下来输入到终端再回车 
> 6. 输入MAC密码 
> 7. 执行完成之后尝试打开 http://wefuture.cqupt.edu.cn/ ,可以打开的话就是进入了内网
